
# Simpleshop-backend

API propia desarrollada para alimentar el frontend `simpleshop-frontend`. 
Reemplaza la API pública original con funcionalidad completa y persistencia real.

📌 Características:
- Autenticación JWT con login y protección de rutas
- CRUD completo para productos, usuarios y listados
- Subida y eliminación de imágenes (almacenadas en servidor)
- Paginación en todas las entidades relevantes (por servidor)
- Base de datos relacional (MySQL/H2) — esquema flexible y listo para migrar

---
> Nota:  
> Este repositorio **no incluye el frontend ni Docker**.
